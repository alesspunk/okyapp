Typography Guidelines

Primary Font: Nunito Sans (Google Fonts)

Fallback Fonts: sans-serif

Line Height: Most styles use 1.43 (143%) for optimal readability

Font Loading: Load font weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (Semi Bold), 700 (Bold), 800 (Extra Bold), 900 (Black)

Responsive Scaling: Scale font sizes proportionally on mobile devices using a base multiplier of 0.875 (87.5%)

About Nunito Sans: Nunito Sans is a well-balanced sans-serif typeface with rounded terminals, providing excellent readability and a friendly, modern appearance.

Icon Token: use icon-large, icon-medium, icon-small, and icon-xsmall for regular icons. Use icon-large-solid, icon-medium-solid, icon-small-solid, and icon-xsmall-solid for solid icons.

Default Icon Variant: all components use fa-regular. Use fa-solid only for the active icon in bottom navigation.

Kit Configuration

This design system uses a Font Awesome 6 Kit loaded via JavaScript. All icons use the new FA6 class syntax.

Kit Script:

<link rel="stylesheet" href="https://ka-p.fontawesome.com/releases/v6.7.2/css/pro.min.css?token=b5a8f622db">

Syntax:

<i class="fa-regular fa-icon-name icon-medium"></i>

Prefixes: fa-regular (default)  |  fa-solid (bottom-nav active only)  |  fa-brands (brand logos)

Token: size and weight are controlled by icon token classes. Keep regular tokens as default and use matching *-solid tokens only for active bottom-nav icons.

Size Reference

Use icon-medium as default. Tokens define sizes (48/24/16/12); style/weight comes from fa-regular or fa-solid.

Sizes Class / Style Usage
icon-xsmall / small / medium / large fa-regular + token classes (12/16/24/48, line-height 100%)
icon-xsmall / small / medium / large-solid Solid tokens preview (12/16/24/48). Use for active bottom-nav icon behavior.
default usage Use regular icons across components. Only active bottom-nav icons switch to fa-solid.

// mobile/iconMedium (default)

// With custom size

1. Buttons

Button.jsx 
import React from 'react';

const Button = ({
  variant = 'primary',
  size = 'medium',
  icon,
  children,
  disabled = false,
  onClick,
  ...props
}) => {
  const buttonStyles = {
    primary: {
      background: '#410088',
      color: '#ffffff',
      border: 'none',
      boxShadow: '0 4px 8px rgba(85, 37, 136, 0.3)',
    },
    secondary: {
      background: '#ffffff',
      color: '#410088',
      border: 'none',
    },
    outlined: {
      background: '#ffffff',
      color: '#410088',
      border: '1px solid #410088',
    },
  };

  const sizeStyles = {
    large: { height: '56px', padding: '0 32px', fontSize: '20px', borderRadius: '44px' },
    medium: { height: '44px', padding: '0 24px', fontSize: '16px', borderRadius: '32px' },
    small: { height: '36px', padding: '0 16px', fontSize: '14px', borderRadius: '20px' },
  };

  const baseStyle = {
    fontFamily: 'Nunito Sans, sans-serif',
    fontWeight: '600',
    cursor: disabled ? 'not-allowed' : 'pointer',
    transition: 'all 0.3s',
    display: 'inline-flex',
    alignItems: 'center',
    justifyContent: 'center',
    gap: '8px',
    opacity: disabled ? '0.5' : '1',
    ...buttonStyles[variant],
    ...sizeStyles[size],
  };

  return (
    <button
      style={baseStyle}
      disabled={disabled}
      onClick={onClick}
      {...props}
    >
      {icon && <i className={`fa-regular fa-${icon}`} />}
      {children}
    </button>
  );
};

// Usage Examples:
<Button variant="primary" size="large">Large Button</Button>
<Button variant="primary" size="medium" icon="check">With Icon</Button>
<Button variant="outlined" size="small" disabled>Disabled</Button>

export default Button;

DropdownField.jsx 
import React from 'react';

// Tokens: border rgba(0,0,0,0.23) · text-sec rgba(0,0,0,0.54) · text-pri rgba(0,0,0,0.87)

const DropdownField = ({
  label,        // floating label shown when value present
  value,        // selected display string
  placeholder, // text when empty
  flagSrc,     // Country variant: circular flag image src
  onClick,
}) => {
  const hasValue = value?.length > 0;
  const showGlobe = !flagSrc && !hasValue; // Country/Empty state

  return (
    <div style={{ position: 'relative', width: '328px' }}>
      <div role="combobox" aria-expanded="false" onClick={onClick}
        style={{
          display: 'flex', alignItems: 'center', gap: '10px',
          height: '56px', borderRadius: '20px', padding: '0 16px',
          border: '1px solid rgba(0,0,0,0.23)', background: 'white', cursor: 'pointer',
        }}
      >
        {showGlobe && <i className="fa-regular fa-globe"
          style={{ color: 'rgba(0,0,0,0.38)', fontSize: '20px', flexShrink: 0 }} />}
        {flagSrc && <span style={{
          width: '24px', height: '24px', borderRadius: '50%',
          overflow: 'hidden', flexShrink: 0,
        }}>
          <img src={flagSrc} alt="" style={{ width: '100%', height: '100%', objectFit: 'cover' }} />
        </span>}
        <span style={{
          flex: 1, fontSize: '16px', fontFamily: 'Nunito Sans, sans-serif',
          fontWeight: hasValue ? '700' : '400',
          color: hasValue ? 'rgba(0,0,0,0.87)' : 'rgba(0,0,0,0.54)',
        }}>
          {hasValue ? value : placeholder}
        </span>
        <i className="fa-regular fa-chevron-down"
          style={{ color: 'rgba(0,0,0,0.38)', fontSize: '18px', flexShrink: 0 }} />
      </div>
      {hasValue && label && (
        <span style={{
          position: 'absolute', left: '18px', top: '-6px',
          background: 'white', padding: '0 4px',
          fontSize: '12px', color: 'rgba(0,0,0,0.54)',
        }}>{label}</span>
      )}
    </div>
  );
};

// Usage:

// Design tokens
// Primary:   #410088  |  Secondary: #18afa5
// Text-sec:  rgba(0,0,0,0.54)  |  Disable: rgba(0,0,0,0.38)

const RadioOption = ({
  label,
  active = false,
  disabled = false,
  flagSrc,
  onDots,
  onChange,
}) => {
  const borderColor = disabled
    ? 'rgba(0,0,0,0.38)'
    : active ? '#410088' : 'rgba(0,0,0,0.54)';

  return (
    <div
      style={{
        display: 'flex',
        alignItems: 'center',
        gap: '16px',
        padding: '14px 8px',
        borderBottom: '1px solid rgba(0,0,0,0.12)',
        cursor: disabled ? 'not-allowed' : 'pointer',
      }}
      onClick={() => !disabled && onChange?.()}
    >
      {/* Radio control */}
      <span style={{
        width: '22px', height: '22px', borderRadius: '50%',
        border: `2px solid ${'$'}{borderColor}`,
        display: 'inline-flex', alignItems: 'center',
        justifyContent: 'center', flexShrink: '0',
      }}>
        {active && <span style={{
          width: '12px', height: '12px',
          borderRadius: '50%', background: '#410088',
        }} />}
      </span>

      {/* Flag — flags/xlarge spec 32×32px circular, aureola only when active */}
      {/* Active aureola: 1px secondary border + 2px white inner gap */}
      {flagSrc && <span style={{
        width: '32px', height: '32px', borderRadius: '50%',
        overflow: 'hidden', flexShrink: '0',
        boxShadow: active ? '0 0 0 2px #fff, 0 0 0 3px #18afa5' : 'none',
      }}>
        <img src={flagSrc} alt=""
          style={{ width: '100%', height: '100%', objectFit: 'cover' }} />
      </span>}

      {/* Label — bold primary when active */}
      <span style={{
        flex: '1', fontFamily: 'Nunito Sans, sans-serif',
        fontSize: '16px',
        fontWeight: active ? '700' : '500',
        color: disabled ? 'rgba(0,0,0,0.38)'
          : active ? '#410088' : 'rgba(0,0,0,0.54)',
      }}>{label}</span>

      {/* 3-dot menu — only when active */}
      {active && <i
        className="fa-regular fa-ellipsis-vertical"
        style={{ color: 'rgba(0,0,0,0.54)', fontSize: '18px', cursor: 'pointer' }}
        onClick={e => { e.stopPropagation(); onDots?.(); }}
      />}
    </div>
  );
};

PromoCode.jsx 
import React from 'react';

export default function PromoCode({ initialCode = '' }) {
  const [code, setCode] = React.useState(initialCode);
  const [isModalOpen, setIsModalOpen] = React.useState(false);
  const [draft, setDraft] = React.useState('');

  const isEntered = code.trim().length > 0;

  function openModal() {
    setDraft('');
    setIsModalOpen(true);
  }

  function closeModal() {
    setIsModalOpen(false);
  }

  function applyCode() {
    const value = draft.trim();
    if (!value) return;
    setCode(value);
    setIsModalOpen(false);
  }

  function clearCode() {
    setCode('');
  }

  return (
    <>
      <div className={`promo-code-molecule ${isEntered ? 'entered' : 'empty'}`}>
        <span className="promo-code-leading" aria-hidden="true">
          <img src="./images/PROMOS@2x.webp" alt="" />
        </span>

        <p className="promo-code-copy">
          {isEntered ? (<><span className="promo-code-prefix">Código Promo:</span> {code}</>) : 'Ingresa el código promocional'}
        </p>

        {isEntered ? (
          <button className="promo-code-action promo-clear" type="button" onClick={clearCode} aria-label="Quitar código">
            <i className="fa-solid fa-circle-xmark icon-medium-solid" />
          </button>
        ) : (
          <button className="promo-code-action promo-open" type="button" onClick={openModal} aria-label="Ingresar código">
            <i className="fa-regular fa-chevron-right icon-medium" />
          </button>
        )}
      </div>

      {isModalOpen && (
        <div className="promo-modal" role="dialog" aria-modal="true">
          <div className="promo-modal-panel">
            <h4 className="promo-modal-title">Ingresa tu código promo</h4>
            <input
              className="promo-modal-input"
              value={draft}
              onChange={(e) => setDraft(e.target.value)}
              placeholder="Ej: verano26"
            />
            <div className="promo-modal-actions">
              <button className="btn btn-outlined btn-medium" type="button" onClick={closeModal}>Cancelar</button>
              <button className="btn btn-primary btn-medium" type="button" onClick={applyCode}>Aplicar</button>
            </div>
          </div>
        </div>
      )}
    </>
  );
}

5. Lists

Two list components added for reusable layouts where atoms stay tokenized and consistent.

List Components

List / PLP (ID: dsvtw): Product listing component (360×422) with 3 rows, 100×100 product image, and right-side action chip (Add0/Add1).

List / Cart (ID: Om56O): Cart list component (307×436) with 16px radius, $Border Main border, and 3 rows with right-side quantity chips.

List/Country Selector: Radio-based country rows with optional flag icon and tokenized text/border colors.

import React from 'react';

const HomeCard = ({
  title,
  brands = [],
  expandable = true,
  onVerMas,
}) => {
  const cardStyle = {
    width: '330px',
    borderRadius: '16px',
    border: '1px solid rgba(0,0,0,0.23)',
    background: '#ffffff',
    boxShadow: '0 2px 10.5px rgba(0,0,0,0.1)',
    display: 'flex',
    flexDirection: 'column',
    gap: '8px',
    padding: '4px 0 16px 0',
  };

  const titleStyle = {
    fontSize: '18px',
    fontWeight: '600',
    color: '#410088',
    fontFamily: 'Nunito Sans, sans-serif',
    padding: '8px 16px 0 16px',
  };

  const gridStyle = {
    display: 'grid',
    gridTemplateColumns: '1fr 1fr',
    gap: '8px',
    padding: '0 8px',
  };

  const brandStyle = {
    display: 'flex',
    flexDirection: 'column',

    gap: '8px',

    borderRadius: '8px',
    padding: '8px',
    border: '1px solid rgba(0,0,0,0.08)',
  };

  return (
    <div style={cardStyle}>
      <div style={titleStyle}>{title}</div>

          <div key={i} style={brandStyle}>

        ))}
      </div>
      {expandable && (
        <div style={{ display: 'flex', justifyContent: 'center', paddingTop: '4px' }}>

  );
};

Usage Guidelines

Button Selection

  • Primary: Use for main actions (Submit, Confirm, Buy Now)
  • Secondary: Use for alternative actions (Cancel, Back, Skip)
  • Outlined: Use for tertiary actions (Learn More, Details, Info)

Size Selection

  • Large: Hero sections, important CTAs, landing pages
  • Medium: Standard forms, dialogs, general UI
  • Small: Compact interfaces, mobile-optimized views, inline actions

Tile Usage

  • Full Tiles: Main service categories on homepage, primary navigation
  • Half Tiles: Secondary services, seasonal promotions, quick links
  • Widgets: Dashboard summaries, quick access panels, informational cards

Accessibility Standards

Color Contrast: All text meets WCAG AA standards

  • Primary text on white: 14.8:1
  • Secondary text on white: 4.5:1
  • Disabled text on white: 3.1:1

Touch Targets: Minimum 44x44px for all interactive elements

Focus States: Visible focus indicators on all interactive components

Labels: Proper labeling for screen readers on all form inputs

Responsive Considerations

  • Base Width: 360px (mobile-first approach)
  • Layout: Vertical stack for mobile devices
  • Spacing: 8px grid system throughout
  • Buttons: Fixed heights, width adapts to content/container
  • Tiles: Maintain aspect ratios across breakpoints
  • Navigation: Fixed at bottom for mobile

Component Naming Convention

The design system follows a hierarchical naming pattern:

Category/Type/Size/State

Examples:

  • Button/Primary/Large/Default
  • Button/Outlined/Medium/Icon
  • Macro/Tile/Food
  • Widget/Daily Offers
  • Input/Phone/Hasvalue
  • Dropdown/Country/Empty
